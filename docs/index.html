What you're looking at

The world

A Kuramoto-style simulator, N ≤ 8 oscillators in the unit square. Each node has a position, a natural frequency, and a fixed-pitch voice; audio is pulse-driven (one short pitched click per zero-crossing). Configs are single YAML files; runs are fully deterministic under (config + seed) so the artifacts you click in the browser are byte-identical to what the CI regenerates.

A regime

A detector-named state the system can sustain for a while — phase-locked, drifting, phase beating, flam, polyrhythmic, dominant cluster, unstable bridge, brittle lock. Each has a deterministic detector that reads the simulator's state and returns fired / silent plus a confidence margin. Labels are ground truth, not model outputs.

The Intervention Atlas

For one baseline run, enumerate every single-node intervention the simulator exposes — ablate_node(k) and nudge_node(k, +0.25 Hz) for each k ∈ [0, N) — re-simulate each one, and rank by how much the detector verdicts move. Score is one line: (#detector flips vs baseline) + |Δ tail_1s_mean_r|. Each row carries a plain-English explanation, so the ranking is readable without opening the JSON.

Hearing the intervention

Each run emits a short audio.wav: one pitched click per pulse, pitched as 180 + 80·ω₀ Hz so nodes are audibly distinguishable. A +0.25 Hz nudge is itself audible — the pitch shifts. The A/B playback strip under the comparison card lets you swap two sync'd audio elements mid-play so you hear the flip without losing position.
